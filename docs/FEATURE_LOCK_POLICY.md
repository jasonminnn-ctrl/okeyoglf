# OkeyGolf OS — 기능 잠금 정책 정리

> 최종 갱신: 2026-03-11 (8-0 준비정리)

---

## 잠금 상태 정의

| 상태 | 코드 | 의미 |
|---|---|---|
| 활성 | `enabled` | 정상 사용 가능 |
| 잠금 | `locked` | UI 노출되나 사용 불가, 업그레이드 안내 |
| 숨김 | `hidden` | UI에서 완전히 제거 |

---

## 멤버십별 기본 정책 요약

### Trial (체험판)
- **활성**: 오늘의 할 일, 체크리스트, 결과 저장, 대시보드 조회
- **잠금**: 기본 생성 기능 대부분, 결과 복사/재생성/내보내기/공유
- **숨김**: 고급 기능 전체 (KPI, 캠페인, 시즌, 채널 등)

### Standard (스탠다드)
- **활성**: 기본 생성 기능, 결과 저장/복사/재생성/내보내기/공유
- **잠금**: 고급 기능, 전담 컨설턴트, 결과 전달
- **숨김**: 없음

### Pro (프로)
- **활성**: 전체 기능 + 전담 컨설턴트 + 결과 전달
- **잠금**: 없음
- **숨김**: 없음

### Enterprise (엔터프라이즈)
- Pro와 동일 + 멀티브랜치/고급 권한 (향후)

## 외부 연동 관련 잠금 정책 (8단계 기준)

| 기능 | 현재 | 8단계 이후 |
|---|---|---|
| AI 생성 (OpenAI 기반) | placeholder (내부 모의 생성) | API 키 등록 시 실제 연결 |
| 아임웹 상품→멤버십 | 미구현 | 운영자 매핑 설정 완료 시 활성 |
| 아임웹 포인트→크레딧 | 미구현 | 운영자 매핑 설정 완료 시 활성 |
| 알림/발송 | placeholder | provider API 키 등록 시 활성 |
| 외부 시장조사 수집 | placeholder | 수집 엔진 연동 시 활성 |

> 원칙: "연결 준비" 상태를 "작동 중"으로 오해하지 않게 명확히 표시.

---

## 준비중 기능 해제 원칙

1. **전부 해제 금지** — 실제 연결된 기능만 활성화
2. **연결 전 단계** → "연결 준비" / "운영자 설정 필요" / "권한 필요" 표시
3. **고객 클릭 시 오해 방지** — 잠금 시 명확한 사유 문구 제공
4. **운영자만 활성화 가능** — /operator에서 override로 제어
